# 総合型選抜テーマバンク（ホワイトスペース企画の在庫）

SKILL.md §1 から参照される在庫。各企画は「中心命題／経営レンズ／哲学レンズ／空白理由」の
4点セット。使う際は必ず話者の一次経験（具体談）で肉付けし、制度事実は要出典確認を付ける。
在庫をそのまま使うと一般論化してホワイトスペース性が死ぬ。

---

## 1. 経営実務の翻訳系（最有望領域）

### 志望理由書＝投資ピッチ
- 命題：「落ちる志望理由書は文章が下手なのではなく、エクイティストーリー（実績→計画→資金使途の接続）が壊れている」
- 経営レンズ：資金調達ピッチの審査で見るポイント／哲学レンズ：「なぜこの大学か」の「なぜ」の種類を区別する（因果・目的・正当化）
- 空白理由：書き方講座勢は文章表現を語り、審査した側の構造は語れない

### 面接＝採用面接＋デューデリジェンス
- 命題：「面接は自己PRの場ではなく、情報の非対称性を解消する相互DDの場」
- 経営レンズ：構造化面接・行動面接（過去の行動事実で将来を予測する）、逆質問の設計
- 哲学レンズ：「本当の自分を見せる」という通念の解体（見せるのは一貫した行動の説明可能性）
- 空白理由：模擬面接勢は想定問答を配るが、面接官側の判断構造は語らない

### 活動実績＝トラックレコード
- 命題：「実績は量ではなく再現性。盛った実績はDD（深掘り質問）で必ず剥がれる」
- 経営レンズ：投資判断でトラックレコードをどう検証するか、盛り実績の剥がし方の実例構造
- 哲学レンズ：「成果」と「経験」の概念区別

### アドミッションポリシー＝IR資料
- 命題：「募集要項は読むものではなく、発行体（大学）の意図を分析するもの」
- 経営レンズ：IR資料・採用要件定義の行間の読み方（書かれた要件と本当の選好のズレ）
- 空白理由：情報系チャンネルは要項を要約するだけで、分析の方法論を渡さない

### 併願戦略＝ポートフォリオ理論
- 命題：「総合型・学校推薦型・一般の併願は、リスク資産の資本配分問題として設計できる」
- 経営レンズ：期待値・分散・撤退基準（サンクコストで出願を続けない）
- 要出典確認：各方式の日程・併願可否は年度で変わるため必ず最新確認

## 2. 哲学の道具箱系

### 「やりたいことがない」の問い直し
- 命題：「『やりたいことは何か』は問いの設計ミス。答えられる問いに作り替えれば志望理由は書ける」
- 哲学レンズ：問いの分解（欲求・適性・持続可能な関心の区別）、「どんな問題なら退屈せず考え続けられるか」への置換
- 経営レンズ：事業ドメイン選定も同じ構造（「何がやりたいか」でなく「どの問題に張るか」）
- 空白理由：キャリア系は「好きを見つけよう」で止まり、問いの設計技術を渡さない

### 募集要項の頻出語を定義し直す
- 命題：「『主体性』『探究心』を定義せずに書類を書くのは、仕様の曖昧な発注書で開発するのと同じ」
- 哲学レンズ：概念の厳密定義（主体性＝指示がない状況での意思決定の履歴、等）
- 経営レンズ：採用要件定義の実務でのバズワード排除

### 「夢を持て」の解体と再構築
- 命題：「夢の強制は選抜上も哲学上も有害。計画は仮説でよい——ただし検証履歴が要る」
- 哲学レンズ：目的論的人生観の吟味／経営レンズ：事業計画も仮説検証で語る（ピボット履歴は減点でなく加点）

## 3. 保護者の意思決定系（CFO領域・完全な空白）

- 「総合型対策塾の料金とインセンティブ構造」——成果報酬・コマ売り・コンサル型の構造比較、情報の非対称性の見抜き方（特定塾の名指し評価はしない。構造のみ）
- 「教育投資のROIをCFOはこう考える」——総合型対策への課金判断のフレームワーク（金額の断定はせず判断軸のみ）
- 「子の書類に親がどこまで関与すべきか」——ガバナンスの比喩（執行と監督の分離）

## 4. 不合格在庫（この路線でやってはいけない企画）

| 却下パターン | 理由 |
|---|---|
| 「◯◯大学の総合型はこう攻略」型の大学別対策 | 一次経験の外。専門塾勢のレッドオーシャンで、制度事実の鮮度リスクも最大 |
| 合格者インタビュー・体験談 | n=1で再現性なし。体験談勢と同じ土俵 |
| 出願日程・倍率速報 | 速報戦は個人が負ける。鮮度切れで資産にならない |
| 「経営者はこう考える」だけの自己啓発論 | 哲学レンズ欠落。意識高い系ビジネス動画に埋没（鉄則2違反） |
| 特定の塾・コンサルの批判 | 誠実性ゲート違反。構造批判はよいが名指しはしない |
