# 塾YouTubeホワイトスポット30分脚本（juku-yt-whitespot-script-jp）

## 鉄則
1. **ホワイトスポットなき脚本は書かない。** 競合が既に擦ったテーマをそのまま書いても再生されない。必ず§1の探索を通し「誰もやっていない×需要がある」交点を特定してから書く。
2. **反常識は結論先出し、根拠は三層。** 「実は逆です」と冒頭30秒で言い切り、データ・一次経験・論理の三層で支える。根拠が一層しかない主張は脚本に入れない（薄い反常識はただの逆張りで、信頼を削る）。
3. **反権威は「制度・慣行・定説」を撃つ。個人は撃たない。** 批判対象は仕組みや業界の常識であって、特定の個人・塾・学校の名指し攻撃ではない。名指しは信頼と法的安全の両方を失う。
4. **30分は「10分×3本の連結」ではなく「1本の推理劇」。** 各ブロックが次の謎（オープンループ）を残す構造で書く。ブロック間に繋ぎがない台本は15分で離脱される。

## §0　入力の確認（不足分だけ聞く。揃っていれば即着手）

毎回の入力から取得するもの：
| 項目 | 例 | 無い場合のデフォルト |
|---|---|---|
| 塾の属性 | 個別指導/集団/オンライン、対象学年、地域 | 「地域密着の中小塾、小中高対応」と仮置きし冒頭で明示 |
| チャンネルの現状 | 登録者数、既存動画の路線 | 新規チャンネル想定 |
| 視聴者ターゲット | 保護者/生徒/塾業界 | **保護者**（塾集客の財布の意思決定者） |
| 話者キャラ | 塾長が顔出し/声のみ 等 | 塾長が一人語り |
| 避けたい話題 | 特定教材の批判NG 等 | なし |

## §1　ホワイトスポット探索（企画工程）

### STEP 1-1　競合地図を描く
WebSearchが使える環境なら「学習塾 YouTube」「教育系YouTuber 人気」「(テーマ候補) YouTube」で実検索し、使えない環境ではユーザー提供情報と下表の定型地図で代替する。参考実例と定番テーマの在庫は `references/theme-bank.md` を読むこと。

教育系YouTubeの主要勢力（レッドオーシャン＝ここと同じ土俵で戦わない）：
| 勢力 | 典型コンテンツ | 飽和度 |
|---|---|---|
| 大手予備校・有名講師系 | 授業解説、科目攻略 | 極高（授業品質では勝てない） |
| 学歴エンタメ系 | 学歴ネタ、クイズ、偏差値企画 | 極高 |
| 勉強法・自己啓発系 | 「〜な勉強法5選」「東大生の習慣」 | 高 |
| 受験情報系 | 大学入試制度、共通テスト速報 | 高 |
| 個人塾長の地域発信系 | 塾紹介、地域の高校情報 | 低〜中（質が低いだけで数はいる） |

### STEP 1-2　ホワイトスポット行列を回す
「テーマ×視聴者×切り口」の三軸を掛け合わせ、競合地図に**存在しない交点**を最低5個列挙する：

- **テーマ軸**：塾業界の裏側／教育費の経済学／内申点・調査書の実態／教育制度の設計思想／保護者の意思決定／模試・偏差値の仕組み／教材ビジネス／不登校・多様な進路
- **視聴者軸**：保護者（主戦場・競合最少）／生徒／塾業界関係者
- **切り口軸**：反常識（定説の反転）／構造暴露（業界のカネと仕組み）／データ検証（通説を数字で潰す）／当事者告白（塾長だから言える一次情報）

**ホワイトスポット判定基準**（3つ全部を満たしたら合格）：
1. 検索・競合地図で同テーマ同切り口の動画が上位に見当たらない
2. 視聴者に切実な悩み・損得が絡む（「面白い」だけでは30分持たない）
3. 話者（塾長）が一次情報で語れる（借り物の知識だけのテーマは深みが出ず、権威側の劣化コピーになる）

### STEP 1-3　企画候補を提示して1本選ぶ
候補5個を「仮タイトル＋反常識の核＋想定視聴者＋ホワイトスポットである理由」の4点セットで提示し、ユーザーに選ばせる。自動で選ぶ指示があれば判定基準の充足度が最も高いものを選ぶ。

## §2　反常識・反権威シナリオ設計（構成工程）

### STEP 2-1　反常識ステートメントを1文で固定する
形式：「みんな【定説X】と思っているが、実は【反転Y】。なぜなら【構造Z】だから。」
この1文が動画の背骨。以降のすべての章はこの1文の証明に奉仕する。証明に寄与しない小ネタは削る。

### STEP 2-2　根拠を三層で用意する
| 層 | 中身 | 例 |
|---|---|---|
| データ | 公的統計・調査・料金相場 | 文科省調査、学習費調査、模試母集団の構成 |
| 一次経験 | 塾長自身の現場エピソード | 「うちの塾で実際にあった〜」 |
| 論理 | 誰の利害でその定説が維持されているかの構造分析 | 「〜が儲かる仕組みだから、その常識は再生産される」 |

数値・調査名は**記憶で断定しない**。検索できる環境では出典を確認し、できない環境では「（要出典確認：〜）」と台本に明示してユーザーに裏取りを促す。捏造された統計は一発でチャンネルの信頼を殺す。

### STEP 2-3　反権威の安全設計（必須ゲート）
- 撃ってよい対象：業界慣行、料金体系の構造、制度設計、「定説」そのもの、自分自身の過去（自虐は最も安全で強い反権威）
- 撃ってはいけない対象：特定の個人・塾・学校・教材の名指し否定、断定的な悪意の帰属（「騙している」ではなく「そういう構造になっている」と言う）
- 事実と意見を話法で区別する：「データ上こうです」と「私はこう考えます」を言い分ける
- 公開前に**hasshin-compliance-jp**の最終チェックを通すことを納品時に必ず案内する

### STEP 2-4　30分の章立て（オープンループ設計）
本論は3幕構成。各幕の終わりに次の謎を置く。幕ごとの謎がないと中盤離脱が起きる。

---

## §3　30分構成テンプレート（時間割）

分速300字換算で全体約8,500〜9,500字。ALWAYS use this exact template:

```
【0:00-1:00 フック】（約300字）
反常識ステートメントの結論を言い切る。視聴者の損得に直結させる
（「知らないと年間◯十万円損する」型）。この動画で何が分かるかを3点予告。

【1:00-3:00 前提共有＋自己開示】（約600字）
定説Xがなぜ常識になっているかを一度「正しく」説明する（藁人形にしない。
定説を丁寧に立てるほど、反転の衝撃が大きくなる）。
話者の立場を開示する（「塾側の人間が言うのだから、ポジショントークを
疑いながら聞いてほしい」——この自己開示自体が反権威の信頼装置）。

【3:00-10:00 第1幕：定説が崩れる瞬間】（約2,100字）
データ層の根拠を提示し、定説Xに最初のヒビを入れる。
幕末オープンループ：「でも、だったらなぜ誰もそう言わないのか？」

【10:00-18:00 第2幕：構造の暴露】（約2,400字）
論理層。誰の利害でこの定説が維持されているかを図解的に語る。
一次経験のエピソードを1〜2本挿入（抽象論が続くと離脱するため、
8分帯と14分帯に具体話を配置）。
幕末オープンループ：「では、私たちはどうすればいいのか？」

【18:00-26:00 第3幕：実践指針】（約2,400字）
視聴者が明日からできる具体行動を3〜5個。ここが薄いと
「文句を言うだけの動画」になり反権威が逆効果になる。
反論処理を1つ入れる（「こう言うと必ず〜という反論が来ますが」）。

【26:00-29:00 まとめ】（約900字）
反常識ステートメントを再掲→3幕の要点を1行ずつ復唱→
「定説を信じるか、構造を見るかはあなたが決めること」型の余韻。

【29:00-30:00 CTA】（約300字）
自塾への導線はここだけ・ソフトに（「概要欄」誘導）。
チャンネル登録理由を「次回予告のオープンループ」で作る。
```

**リテンション装置の配置規則**：
- 4〜6分間隔でパターン割り込みを入れる（問いかけ／具体エピソード／「ここからが本題」宣言）
- 各幕の冒頭で「いま全体のどこか」を一言で現在地表示する（30分動画は迷子離脱が最大の敵）

---

## §4　台本執筆規約（文体）

- 一人語りの話し言葉で書く。一文は50字以内目安。書き言葉の漢語連結（「〜であるがゆえに」）は禁止
- 画面指示は台詞と分離して `［テロップ：〜］［図解：〜］` 形式で行頭に置く
- 数字は必ず比較対象とセットで出す（「30万円」ではなく「公立中3年間の塾代の約半分にあたる30万円」）
- 断定と留保を使い分ける：データがある箇所は断定、解釈の箇所は「私は〜と見ています」
- 各幕の見出しに目安時間と目安字数を残す（収録時のペース管理用）

## §5　納品形式

1. **企画書パート**：選定したホワイトスポット、反常識ステートメント、三層根拠の一覧、（要出典確認）項目のリスト
2. **台本本体**：§3テンプレート準拠のフル台本（約9,000字）
3. **付帯情報**：タイトル案3本（反常識型／損得型／疑問型）、サムネ文言案3本、概要欄文面
4. **公開前の案内**：「公開前にhasshin-compliance-jpで法令・炎上チェックを通してください」と必ず一言添える

## 例（Input→Output）

**Input**（現実の雑な発話）：
「YouTubeやりたいんだけどネタがない。他の塾がやってないやつで、ちょっと攻めた30分のやつ作ってよ。うちは個別指導で中学生メイン」

**Output**（要旨）：
1. §1探索を実行し、候補5本を提示。例：
   - 「塾長が告白：『個別指導なら成績が上がる』は半分ウソです」（反常識×当事者告白×保護者向け。個別指導の効果は生徒タイプ依存という一次情報で語れる、競合上位に不在）
   - 「模試の偏差値のカラクリ——お子さんの偏差値が塾で上がって見える構造的理由」（データ検証×保護者向け）
   - ほか3本
2. ユーザーが1本目を選択→反常識ステートメント固定：「みんな『個別指導は手厚いから上がる』と思っているが、実は上がる子と上がらない子は入塾前に決まっている。なぜなら個別指導の効果は指導形態ではなく家庭学習の設計で決まるから。」
3. §3テンプレートで約9,000字の台本＋タイトル案＋サムネ案を納品し、hasshin-compliance-jpでの最終チェックを案内。
